# Installation

There are two ways to install the Magento 2 Hyvä Jewellery Theme: by using the release package or by installing it through Composer.

## Option A — Install from the release package

Ask your Magento technical team to copy the theme files into the Magento installation and run the normal Magento setup steps.

Use this option when you received a zip package or a direct theme package.

## Option B — Install with Composer

Use this option when the theme package is available in your Composer repository.

Your Magento technical team can install the package, run Magento setup, and deploy store assets.

## After installation

1. Clear Magento cache.
2. Open Magento Admin.
3. Go to **Content -> Design -> Configuration**.
4. Apply **Adornment Jewelry Theme** to the correct store view.
5. Open **Content -> Homepage Configurator -> Sections**.
6. Confirm the default homepage sections are available.

Continue with [Activate the Theme](/activation.html).
