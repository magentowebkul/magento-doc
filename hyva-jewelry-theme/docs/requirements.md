# Requirements

Before installing the Adornment Jewelry Theme, make sure your environment meets the following requirements.

## Server & platform

| Component | Minimum | Recommended |
| --- | --- | --- |
| Magento | 2.4.x | 2.4.6+ |
| PHP | 8.1 | 8.1 / 8.2 |
| Hyva Theme | `hyva-themes/magento2-default-theme` | latest stable |
| MariaDB / MySQL | 10.4 / 8.0 | 10.6 / 8.0 |
| Elasticsearch / OpenSearch | 7.x / 2.x | latest |
| Composer | 2.x | 2.x |
| Node.js | 16.x | 18.x or 20.x |
| npm or yarn | any | npm |

## Required Hyva setup

Hyva must be installed before this theme is activated. If Hyva is not installed yet, install Hyva first and confirm your checkout setup is working.

The theme is made to work as a Hyva-based storefront theme.

## Store setup checklist

- Magento Admin access is available.
- Hyva is installed and active.
- The store can clear cache and publish storefront assets.
- Product images, categories, and sample products are ready.
- A technical admin is available for installation if needed.

## Browser support

The storefront targets the current versions of Chrome, Firefox, Safari and Edge. It is mobile-first and responsive down to small phones.
