# Header & Footer Settings

Beyond the menu, the builder controls the header announcement, logo, and footer content.

## Header settings

| Field | Purpose |
| --- | --- |
| Announcement text | Small bar above/within the header (e.g. "Complimentary shipping on orders over $150") |
| Announcement CTA | Label for an optional announcement button/link |
| Logo image | Store logo shown in the header |

## Footer settings

| Field | Purpose |
| --- | --- |
| Footer description | Short brand/about blurb in the footer |
| Contact email | Shown in the footer contact block |
| Contact phone | Shown in the footer contact block |

![Header and footer settings in Magento Admin](/screenshots/header-footer-settings.png)

## After saving

After changing header or footer text, click **Save Configuration**. If the old content still shows, refresh the storefront and clear Magento cache from Admin if needed.
