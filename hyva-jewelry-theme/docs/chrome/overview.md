# Header, Footer & Mega Menu Builder

The **Header & Footer Builder** lets you manage the store navigation, mega menus, header announcement, logo, and footer from Magento Admin.

Open it from **Content → Homepage Configurator → Header & Footer Builder**.

![Magento Admin menu path for Header and Footer Builder](/screenshots/header-footer-builder-menu.png)

![Mega menu on the storefront](/screenshots/mega-menu.png)

## What it controls

Use this builder to update these storefront areas:

| Area | What you edit |
| --- | --- |
| **Menu structure** | Add categories, pages, products, collections, and custom links. Reorder them by drag-and-drop. |
| **Item settings** | Change the label, link, description, visibility, submenu style, and promo images for a menu item. |
| **Header settings** | Announcement text, announcement link, and store logo. |
| **Footer settings** | Description, contact email, contact phone. |
| **Agent Readiness** | Optional settings for AI and search discovery. |

::: warning Save before checking the storefront
Previewing in the builder does not update the live storefront. Click **Save Configuration** after making changes.
:::

## Workflow at a glance

1. Add categories, pages, products, or custom links to the **Saved Menu Structure**.
2. Drag to reorder and nest items into groups.
3. Click a row to edit its label, link, submenu style, and promo images.
4. Set header announcement, logo, and footer text.
5. (Optional) Configure **Agent Readiness**.
6. Click **Save Configuration**.
