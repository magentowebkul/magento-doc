# Magento 2 Hyvä Jewellery Theme

The Magento 2 Hyvä Jewellery Theme is made for jewellery and fashion accessory stores. It gives your store a clean homepage, simple navigation, product listing pages, product detail pages, cart, checkout, account pages, wishlist pages, contact page, and a branded 404 page. It is also Agent Ready, so the storefront can work well for modern AI browsing checks and agent-assisted shopping flows.

This guide explains what to check first, how to activate the theme, and where to manage the main storefront areas.

## Agent Ready

The theme is built on Hyvä for a fast storefront experience. It is prepared for agent browsing checks, so AI tools can understand the page structure, read key storefront content, and move through shopping pages more easily.

The example check below shows strong desktop performance and a passing agentic browsing result for the demo storefront.

## Before you start

Make sure the store has the basic data needed for a good preview:

1. Add active categories and include them in the menu.
2. Add products with names, prices, images, and stock.
3. Set the default store view, language, and currency.
4. Check that Hyvä is installed and active on the Magento project.
5. Keep the Magento cache ready to refresh after changing the theme.

## Getting started

Follow these pages in order:

1. Read the [Requirements](/requirements.html).
2. Complete the [Installation](/installation.html).
3. Activate the theme from [Activate the Theme](/activation.html).
4. Open the storefront and check the homepage, category page, search page, product page, cart, and checkout.
5. If something does not look right, open [Troubleshooting](/help/troubleshooting.html).

## After activation

After the theme is active, check these storefront areas:

| Store area | What to check |
| --- | --- |
| Homepage | Hero banner, category carousel, new arrival products, and featured products. |
| Header | Logo, menu, search box, customer account link, wishlist, compare, and cart. |
| Footer | Newsletter, store links, social links, and payment icons. |
| Category page | Product grid, filters, sorting, pagination, and product cards. |
| Search page | Search field, result count, filters, pagination, and product cards. |
| Product page | Product images, price, options, quantity, add to cart, wishlist, and reviews. |
| Cart and checkout | Cart drawer, cart page, checkout steps, address forms, shipping, payment, and order success page. |
