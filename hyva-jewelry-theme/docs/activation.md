# Activate the Theme

After the theme files are installed, assign the theme to your store view from Magento Admin.

## Apply the theme in Magento Admin

1. Go to **Content -> Design -> Configuration**.
2. Find the store view you want to theme and click **Edit**.
3. In the **Default Theme** section, set **Applied Theme** to **Adornment Jewelry Theme**.
4. Save the configuration.

![Magento Admin Design Configuration showing Adornment Jewelry Theme applied](/screenshots/design-configuration-adornment-theme.png)

## Verify on the storefront

Open the storefront and check the homepage, header, footer, category page, product page, cart, and checkout.

## Assign the Homepage Configurator to the homepage

The theme includes a ready homepage builder. After activation, open **Content -> Homepage Configurator -> Sections** and check the homepage sections.

If the old homepage still shows, clear Magento cache and refresh the storefront.
