# Category Grid

The **Category Grid** section helps shoppers open the right category quickly. Use it for items like Rings, Earrings, Necklaces, Bracelets, Watches, or Gifts.

You can show categories as a grid or as a slider.

## Add a Category Grid

1. Go to **Content -> Homepage Configurator -> Sections**.
2. Click **Add New Section**.
3. Enter a **Title**, for example `Shop by Category`.
4. Set **Section Type** to **Category Grid**.
5. Set **Enabled** to **Yes**.
6. Use a **Sort Order** such as `20`, so it shows below the hero banner.
7. Fill the **Category Grid Configuration** fields.
8. Click **Save**.

![Magento Admin Edit Section page for Category Grid](/screenshots/category-grid-edit-general.png)

## Main settings

| Setting | What it does |
| --- | --- |
| **Display Mode** | Choose **Grid View** for fixed tiles, or **Slider View** for a moving row. |
| **Category Source** | Choose **Select from Catalog** to use Magento categories, or **Manual** to add your own tiles. |
| **Select Categories** | Choose categories from your catalog when using catalog source. |
| **Number of Items to Show** | Limit how many tiles show. Use `0` to show all selected items. |
| **Subtitle** | Small text under the section title. |

![Magento Admin Category Grid main settings](/screenshots/category-grid-main-settings.png)

## Layout settings

| Setting | What it does |
| --- | --- |
| **Columns Desktop** | Number of category tiles per row on desktop. |
| **Columns Mobile** | Number of category tiles per row on mobile. |
| **Image Shape** | Choose Square, Rounded, or Circle. |
| **Show Product Count** | Shows how many products are in the category. |
| **Show Image Overlay** | Adds a soft overlay on the image so text is easier to read. |

![Magento Admin Category Grid layout settings](/screenshots/category-grid-layout-settings.png)

## Manual category tiles

Use **Manual** when you want full control over image, label, and link.

Click **Add Category**, then fill:

| Field | What to enter |
| --- | --- |
| **Category Image** | Upload a clear image. Maximum file size is **2 MB**. |
| **Label** | Category name, such as `Rings`. |
| **URL** | Category page link, such as `/rings.html`. |

![Magento Admin Category Grid manual category items](/screenshots/category-grid-manual-items.png)

## View All button

Turn on **Show "View All" Button** if you want one button below the category section.

Use **View All URL** for the link. Example: `/jewellery.html`.

## Good practice

- Use 4 to 6 categories on the homepage.
- Keep labels short.
- Use the same image style for all category tiles.
- Check mobile view after saving.
