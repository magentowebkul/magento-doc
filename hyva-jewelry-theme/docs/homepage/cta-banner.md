# CTA Banner

The **CTA Banner** section asks shoppers to take one clear action. Use it for booking an appointment, visiting a store, contacting support, or opening a special collection.

CTA means **call to action**.

## Add a CTA Banner

1. Go to **Content -> Homepage Configurator -> Sections**.
2. Click **Add New Section**.
3. Enter a **Title**, for example `Private Styling Appointment`.
4. Set **Section Type** to **CTA Banner**.
5. Set **Enabled** to **Yes**.
6. Use a **Sort Order** so it appears in the right place on the homepage.
7. Fill the **CTA Banner Configuration** fields.
8. Click **Save**.

## Main settings

| Setting | What it does |
| --- | --- |
| **Background Image** | Upload a wide image. Recommended size is **1920 x 600 px**. Maximum file size is **4 MB**. |
| **Overlay Color** | Color placed above the image. Example: `rgba(0,0,0,0.45)`. |
| **Heading** | Main CTA text. |
| **Subheading** | Short text below the heading. |
| **Button Label** | Button text, such as `Book Appointment`. |
| **Button URL** | Link for the button. |
| **Button Style** | Choose **Filled** or **Outline**. |
| **Text Alignment** | Choose **Left**, **Center**, or **Right**. |

## Good practice

- Use one action only.
- Use a darker overlay on bright images.
- Keep the button label short.
- Link the button to a real page, not `#`.
