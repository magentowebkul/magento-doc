# Luxury Grid

The **Luxury Grid** section shows premium cards. Use it for luxury collections, special materials, shopping paths, or jewellery services.

## Add a Luxury Grid

1. Go to **Content -> Homepage Configurator -> Sections**.
2. Click **Add New Section**.
3. Enter a **Title**, for example `Luxury Collections`.
4. Set **Section Type** to **Luxury Grid**.
5. Set **Enabled** to **Yes**.
6. Use a **Sort Order** so it appears in the right place on the homepage.
7. Fill the **Luxury Grid Configuration** fields.
8. Click **Save**.

## Main settings

| Setting | What it does |
| --- | --- |
| **Subtitle** | Small text below the section title. |
| **Columns Desktop** | Choose **2**, **3**, or **4** columns on desktop. |
| **Luxury Items** | Add the cards that will show in the grid. |

## Item fields

Click **Add Item**, then fill these fields.

| Field | What to enter |
| --- | --- |
| **Item Image** | Upload the card image. Maximum file size is **4 MB**. |
| **Label** | Card title, such as `Gold Essentials`. |
| **URL** | Link for the card. |
| **Badge Text** | Optional badge, such as `New` or `25% Off`. |
| **Link Text** | Small link text, such as `Shop Now`. |

## Good practice

- Use the same image shape and lighting for all cards.
- Use short labels.
- Use badges only when they add value.
- Keep the grid focused on premium shopping paths.
