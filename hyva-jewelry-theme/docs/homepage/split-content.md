# Split Content

The **Split Content** section shows an image on one side and text on the other side. Use it for a brand story, a featured collection, a service message, or a short jewellery promise.

## Add Split Content

1. Go to **Content -> Homepage Configurator -> Sections**.
2. Click **Add New Section**.
3. Enter a **Title**, for example `The Modern Heirloom Collection`.
4. Set **Section Type** to **Split Content**.
5. Set **Enabled** to **Yes**.
6. Use a **Sort Order** so it appears in the right place on the homepage.
7. Fill the **Split Content Configuration** fields.
8. Click **Save**.

## Content settings

| Setting | What it does |
| --- | --- |
| **Image** | Upload the section image. Recommended size is **800 x 900 px**. Maximum file size is **4 MB**. |
| **Image Position** | Choose **Left** or **Right**. |
| **Badge Text** | Optional small badge, such as `New` or `Signature`. |
| **Heading** | Main heading. If empty, the section title is used. |
| **Body Text** | Short paragraph for the section. |
| **Button Label** | Button text, such as `Read Our Story`. |
| **Button URL** | Link for the button. |

## Good practice

- Keep the body text short.
- Use one clear message.
- Use **Image Position** to balance the page when you have many sections.
- Check mobile view after saving.
